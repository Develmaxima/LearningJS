camelCase - Стиль написания составных слов (кемелКэйс)
---------------------------------------------------------------------
camelCase должен начинаться со строчной буквы,
а первая буква каждого последующего слова должна быть заглавной.
Все слова при этом пишутся слитно между собой.

Примитив JS
=============
string
boolean
number
undefined
null (typeof null -вернет object и это косяк)

Оператор сравнения, рекомендуется использовать ===
==================
= проверяет по 1 параметру
== проверяет по двум параметрам
=== проверяет по трем параметрам , учитывая тип данных
